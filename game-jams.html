My First Jam - FT Game Jam 2013

FT Game Jam is an annual team-based game jam held in the Faroe Islands. The first time it was held was in February 2013, which was also the first time I participated in a game jam. I would continue to participate in this jam until I moved to Denmark for my studies.

My First Released Game and Failing Faster

In 2013 I made my first proper game and released on Newgrounds. It was really bad. I had spent so much time making the characters and the art, only to end up with a terrible game because I had spent no time making the actual gameplay good.

I decided I would make a lot of smaller games so I could learn how to make a good game, rather spending so much time on making content for a terrible game. This is also part of why I, especially in 2015 - 2016, participated in a lot of game jams.
